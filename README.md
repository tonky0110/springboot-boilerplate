1. java 설치: 
    brew tap adoptopenjdk/openjdk
    brew cask install adoptopenjdk8
    java -version
    brew install maven
    springboot

2. STS 설치: 
    brew cask install sts
3. spring 설치:
    brew -v
    brew tab pivotal/tap
    brew tap pivotal/tap
4. springboot 설치:
    brew install springboot
    spring --version
